cert-gen
========

Generador de certificados y diplomas en pdf bastante básico y seguramente lleno de bugs.

Uso
===================

Permite generar documento(s) pdf a partir de un diseño (preferentemente .png) y un listado de nombres (.txt).


Complementos usados
===================

- Bootstrap v3.2.0
- Jquery v1.10.2
- FPDF v1.7
